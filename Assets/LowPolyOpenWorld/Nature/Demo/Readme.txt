To use the demo scene, you need to do:

1. Set color space to Gamma (Edit -> Project Settings -> Player -> Other Settings -> Color Space)

2. Download and install this assets:
- STANDART ASSETS
https://assetstore.unity.com/packages/essentials/asset-packs/standard-assets-32351
- LEGACY IMAGE EFFECTS
https://assetstore.unity.com/packages/essentials/legacy-image-effects-83913
